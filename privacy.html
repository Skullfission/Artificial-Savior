Privacy Policy

Artificial Savior · Last updated: 2026-04-22

Overview

Artificial Savior is a single-player browser game created by Bruce Beerman. The game is designed to be privacy-respecting: it does not require an account, does not collect personal data, and does not transmit any information to a server.

Data we collect

We do not collect any personal information. The game does not include analytics, advertising, telemetry, or tracking of any kind.

Data stored on your device

  • High scores: Your local high-score list (score and three-letter initials you enter) is saved in your browser's localStorage under the key artificialSaviorHiscores. This data never leaves your device.
  • Offline cache: Game assets (HTML, scripts, images, audio, level data) are cached by the service worker so the game can run offline. This is standard Progressive Web App behavior and contains no personal data.

You can clear both at any time by clearing site data for this app in your browser settings, or uninstalling the app.

Permissions

The game requests no special permissions. It does not access your camera, microphone, location, contacts, files, or any other device capability.

Third-party services

The web version is hosted on GitHub Pages. GitHub may record standard web request logs (IP address, timestamp, user agent) in accordance with GitHub's privacy statement. No other third-party services are used.

Children

Artificial Savior does not knowingly collect information from children. Because no data is collected at all, the game is safe to use regardless of age.

Changes to this policy

If this policy changes, the updated version will be posted at this same URL with a new "Last updated" date.

Contact

Questions about this policy can be directed to the project maintainer via the public repository: github.com/Skullfission/Artificial-Savior.
